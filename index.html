TimeTracker

A simple utility for keeping track of time.

View on GitHub Download .zip Download .tar.gz

TimeTracker Wiki

  • Ever struggle with keeping track of how long you've been working on a project?
  • Does your work use an outdated and poorly designed time clock system?
  • Wondering how little that one person in the group has actually contributed?

TimeTracker will do away with these archaic problems.

Features

  • Quickly add a new user with just a name and a 4-digit PIN
  • Clock in/out with just your PIN
  • View a report with a complete log of all your time

Database Schema

USER
-name: User's name
-pin: User's 4-digit PIN used to securely clock-in/out
-startDate: Date User joined TimeTracker
-lastClockIn: Time of User's most recent clock-in
-totalTime: Count of User's total time worked

Future Work

  • Scale it to an enterprise level with employers and employees
  • Allow users to edit log entries
  • Implement analytics to allow users to see their activity and that of others
