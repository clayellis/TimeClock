Description of our project

TimeTracker is a web app that allows users to keep a log of the hours they have spent working.

  • Ever struggle with keeping track of how long you've been working on a project?
  • Does your work use an outdated and poorly designed time clock system?
  • Wondering how little that one person in the group has actually contributed?

TimeTracker will do away with these archaic problems.

Features

  • Quickly add a new user with just a name and a 4-digit PIN
  • Clock in/out with just your PIN
  • View a report with a complete log of all your time

Database Schema

USER
